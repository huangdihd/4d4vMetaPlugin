> A specialized MetaPlugin for **4d4v.org**, built on the [xinbot](https://github.com/huangdihd/xinbot) framework.

English / [简体中文](README_zh.md)

---

## Features
- **ViaVersion Integration**: Cross-version protocol support for enhanced compatibility.
- **Multi-version Support**: Seamless communication between different Minecraft versions.
- **Protocol Translation**: Automatic packet translation for version compatibility.

---

## Quick Start

1) Download
   Obtain the latest `4d4vMetaPlugin.jar` from the [Releases](https://github.com/huangdihd/4d4vMetaPlugin/releases) page.

2) Install
   Place the JAR file into the `plugin` directory of your xinbot installation. Java 17 or newer is required.

3) Run
   Start xinbot. The plugin will be loaded automatically to handle the 4d4v.org protocol specifics.
